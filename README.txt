Данный репозиторий представляет собой клиентскую часть web приложения, серверная часть которого описана в репозитории FirstLab.
Работает только в связке с backend'ом из репозитория FirstLab. Имеет графический web интерфейс.
Предоставляет следующие возможности:
	-Доступ ко всем элементам находящимся в SERV_DIR/MainFolder
	-Создание пустой директории
	-Удаление каталога и всех вложенных элементов, если они имеются
	-Скачивание файлов.
Клиентская работа вне SERV_DIR/MainFolder не допусткается.
При запросе к корневой папке (SERV_DIR) автоматически откроется index.html и произойдет перенаправление к SERV_DIR/MainFolder.
Удаление MainFolder так же не допускается.
Для использования всех возможностей реализованы интуитивно понятные окна диалога с пользователем.

Для корректной работы приложения необходимо все содержимое репозитория поместить в католог, адресс которого указан в переменной SERV_DIR(см.FirstLab)
